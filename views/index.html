IP Location API

Coded by Autumn Bullard

An API that gets the longitude and latitude for the user's IP address and any other entered IP address.

View the source code for this project.

Example Usages

Get the location for your IP address

To get the coordinates for your IP address, go to the path /api/ip/me.

Get the location for a custom IP address

To get the coordinates for other IP addresses, go to the path /api/ip and then add a parameter of the IP address. For example, if I wanted to get the coordinates for the IP address 72.14.192.0, I would type /api/ip/72.14.192.0.
